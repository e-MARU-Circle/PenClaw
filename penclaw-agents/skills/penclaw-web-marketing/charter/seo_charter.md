## 戦略方針（4層）

| 層 | スタンス | 主担当 |
| --- | --- | --- |
| 指名検索 | 守りに徹する。現状の1位を維持。新規施策不要 | マコト |
| 一般地域語 | オーガニックでは勝負しない。**MEOが本丸**。リン・ナナと連携 | リン／ナナ＋マコト補佐 |
| コンテンツSEO | 自費誘導記事のリライトが最優先 | マコト＋ナナ薬機法監修 |
| **AI検索（v1.2新設）** | 引用獲得が目的。CV率23%の優良チャネルを月50流入まで育てる | マコト＋ナナ＋ケン（権威性） |

理由：
- 4月Search Console実データで、一般地域語はオーガニック上位でもクリック0が複数。マップパックに吸われている構造のため、SEO単独投資は非効率と判断
- ChatGPT流入のCV率23%は、通常検索の2倍以上。AI検索は「具体的に調べた上でクリックする」高関心ユーザー層。早期に布石を打つ価値が高い

### P-1（LP統合）の詳細メモ

- 現状：`kofu.emasika.jp` がサブドメインで自費系LPを抱えている。デザインも各LPで統一感がない
- 目的：(1)ドメインパワーの集約、(2)ユーザー導線統一、(3)管理工数削減、(4)自費マトリクス20本のSEO底上げ
- 注意事項：
  - 旧URLから新URLへの **301リダイレクト必須**（SEO評価の引き継ぎ）
  - 各LPの薬機法・医療広告ガイドライン適合チェック（ナナ）
  - サイトマップ更新、Search Consoleで再申請
  - 内部リンク張り替え
- 段階：
  1. 現状LP棚卸し（マコト）
  2. 統一デザインテンプレート設計（マコト＋先生承認）
  3. 1本ずつ移行＋301設定（マコト）
  4. インデックス確認（マコト、Search Console）

### AI検索戦略の詳細メモ（P-3a / P-5 / P-6 / P-7 共通）

**AIに引用されやすいコンテンツの要件（マコト原則）：**

1. **質問→端的な回答→詳説** の3層構造（FAQ形式の延長）
2. **数字・具体名・引用元** を必ず含む（権威性のシグナル）
3. **見出しは検索クエリそのもの**（「拍動痛とは何か」「インプラントの平均寿命は」）
4. 1段落100-150字、段落間に空行（AIが切り出しやすい構造）
5. **専門家プロフィール紐付け**（執筆者：江間秀明 副院長、矯正認定医など）
6. **構造化データ**（FAQPage / MedicalProcedure / Person）を必ず付ける

**ChatGPT/Claude/Perplexity が学習しているソースに乗せる戦略：**
- Wikipedia編集（医院記事ではなく、関連医療トピックでの権威記述貢献）
- 医療系メディア寄稿（caloo.jp / medicaldoc.jp など既に流入実績ありの媒体）
- Google ナレッジグラフ最適化（GBP情報と Web上の医院記述の整合性）

**手動チェック運用（月次）：**
固定追跡28キーワードのうち、以下5つを毎月各AI検索で叩いて、当院引用の有無を記録（action_log.md）：
- 甲府市 歯医者
- 甲府市 矯正歯科
- 甲府市 インプラント
- 拍動痛
- ブラックステイン

各AI（ChatGPT / Claude / Perplexity / Gemini / Google AI Overview）で実施。
